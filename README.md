# 🚀 Justine Todo List

A sleek, modern, and high-performance Todo application built with React, Vite, and Framer Motion. Featuring glassmorphism, smooth animations, and persistent storage.

## 🛠 Features

- **Glassmorphism UI**: Beautiful, translucent design with vibrant gradients.
- **Animations**: Fluid transitions using Framer Motion.
- **Persistence**: Automatically saves your tasks to LocalStorage.
- **Responsive**: Works perfectly on mobile and desktop.
- **Deployment Ready**: Optimized for Vercel and GitHub.

## 🚀 How to Deploy

### Deploy to GitHub
1. Create a new repository on GitHub.
2. Initialize git in this folder:
   ```bash
   git init
   git add .
   git commit -m "Initial commit"
   git branch -M main
   git remote add origin YOUR_GITHUB_REPO_URL
   git push -u origin main
   ```

### Deploy to Vercel
1. Go to [Vercel](https://vercel.com).
2. Click **New Project**.
3. Import your GitHub repository.
4. Vercel will automatically detect the Vite settings. Click **Deploy**.
